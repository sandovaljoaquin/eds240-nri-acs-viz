# Visualizing FEMA NRI Data

The purpose of this repository is to create data visualizations that show (1) how FEMA National Risk Index scores for counties in California compare to those in other states and (2) how climate hazard risk exposure varies across racial/ethnic groups in California. 

Content: 

```
├── data
│   ├── ACS-1yr-2023-county-race-ethnicity.csv
│   └── National_Risk_Index_Counties_807384124455672111.csv
├── eds240-nri-acs-viz.Rproj
├── HW2.html
├── HW2.qmd
├── HW3.html
├── HW3.qmd
└── README.md
```

Data Access: 

The data used in this repository can be found on the FEMA Resilience Analysis & Planning Tool after selecting the `National Risk Index Counties layer`. It can also be accessed with this link: https://drive.google.com/file/d/1yoavLSBmJIMjVY9fQrT_rg8u6C-3v_v8/view

The `ACS-1yr-2023-county-race-ethnicity.csv` can be found here: https://drive.google.com/file/d/1gS-YlVZwm9NPSFrnGcmG6jooPeCyFjp9/view
